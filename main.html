01

Pengertian HTML

HTML (Hyper Text Markup Language) merupakan bahasa Markup standar yang digunakan untuk membuat dan menyusun halaman website.

  • HTML bukan bahasa pemrograman.
  • HTML tidak bisa berpikir, menghitung, atau mengambil keputusan.
  • Tugasnya HTML: mengatur struktur konten di halaman web.
  • HTML menggunakan tag untuk menandai elemen-elemen di halaman web.
02

Sejarah HTML

HTML adalah singkatan dari HyperText Markup Language, yaitu bahasa markup standar yang digunakan untuk membuat dan menyusun halaman web. HTML bukan bahasa pemrograman, melainkan bahasa penanda (markup) yang berfungsi untuk mengatur struktur konten pada sebuah website.

Awal Mula HTML

HTML pertama kali dikembangkan oleh Tim Berners-Lee pada tahun 1991. Saat itu, Tim Berners-Lee bekerja di CERN (Organisasi Riset Nuklir Eropa). Tujuan utama HTML adalah untuk memudahkan para ilmuwan berbagi dokumen dan informasi melalui jaringan internet.

HTML 2.0 (1995)

Versi standar resmi pertama yang dipublikasikan oleh IETF.

HTML 3.2 (1997)

Dikembangkan oleh W3C, menambahkan dukungan tabel dan script.

HTML 4.01 (1999)

Versi yang sangat populer. Pemisahan antara konten dan presentasi dengan CSS.

HTML5 (2014)

Revolusi besar! Dukungan multimedia, canvas, semantic tags, dan aplikasi web modern.

03

Fungsi HTML

HTML berfungsi untuk membuat dan menyusun struktur halaman website agar bisa ditampilkan oleh browser.

  • Menampilkan teks (judul, paragraf)
  • Menampilkan gambar
  • Membuat tautan (link)
  • Menyusun elemen-elemen di halaman web
  • Mengintegrasikan multimedia (video, audio)
  • Membuat form input (login, pendaftaran)

Cara Kerja HTML

  1. Developer menulis code HTML
  2. File disimpan dengan ekstensi .html
  3. Browser membuka file HTML
  4. Browser membaca dan menerjemahkan tag
  5. Konten ditampilkan sebagai halaman web
